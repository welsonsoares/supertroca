# WS Super Troca de Cromos

Página de álbum de figurinhas para a Copa do Mundo 2026.

## Arquivo principal

- `super-troca-cromos V3beta.html`
- Pode ser usado diretamente no navegador
- Para publicar no GitHub Pages, renomeie para `index.html`

## Funcionalidades

- Lista interativa de figurinhas faltantes
- Controle de repetidas para troca
- Filtro por sigla do time
- Seleção de figurinhas para marcar como recebidas
- Popup animado ao completar um time
- Estatísticas de progresso e times com 80%+ completos
- Modal de scanner e controle de trocas
- Persistência no `localStorage`

## Como usar

1. Abra o arquivo `super-troca-cromos V3beta.html` no navegador.
2. Digite seu nome e comece a usar o álbum.
3. Use a pesquisa por sigla de time para encontrar seleções.
4. Clique nas figurinhas faltantes para marcar seleção e adicionar ao álbum.
5. Use o filtro de times para visualizar apenas as seleções desejadas.

## Publicação no GitHub Pages

1. Crie um repositório público no GitHub.
2. Faça upload do arquivo `super-troca-cromos V3beta.html`.
3. Renomeie-o para `index.html` para facilitar o acesso.
4. No repositório, acesse `Settings` > `Pages`.
5. Escolha a branch `main` ou `master` e a pasta `/root`.
6. Salve e navegue até `https://seu-usuario.github.io/seu-repositorio/`.

## Observação

A busca e o filtro funcionam sempre pela sigla do time (por exemplo: `BRA`, `ARG`, `FRA`). Os nomes completos dos times são exibidos apenas para o usuário.
