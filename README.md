# supertroca
uper Troca de Cromos: Álbum Copa 2026 ⚽  Web app otimizado para Android que gerencia suas figurinhas da Copa do Mundo. Conta com sistema inteligente de controle de faltantes e repetidas, integração nativa com WhatsApp para compartilhamento de listas e um Scanner de Cromo via IA que lê o verso da figurinha pela câmera do celular!
